# QCM Statistiques — Application Streamlit

Application interactive de quiz sur les **statistiques et probabilités**, développée avec Streamlit.

---

## Fonctionnalités

- Quiz interactif sur 2 thèmes : **Probabilités** et **Statistiques descriptives**
- Filtrage des questions par thème via la sidebar
- Questions à choix unique avec explication de la bonne réponse
- Suivi du score en temps réel via `st.session_state`
- Questions aléatoires sans répétition
- Score final affiché à la fin du quiz

---

## Thèmes couverts

**Probabilités**
- Loi des grands nombres
- Calcul de probabilités simples (tirage de cartes, etc.)

**Statistiques descriptives**
- Médiane, moyenne, mode
- Écart-type et dispersion
- Interprétation des mesures statistiques

---

## Lancement

```bash
pip install streamlit
streamlit run app.py
```
